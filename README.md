# Label Encoding for Categorical Data

## 📌 Project Overview
This project demonstrates how to convert categorical data into numerical form using Label Encoding. Each unique category in a column is assigned a numerical value, making the dataset suitable for machine learning algorithms.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 📊 Features
- Identifies categorical columns in the dataset
- Applies Label Encoding
- Converts categorical values into numerical labels
- Prepares dataset for machine learning models

## 🔧 Method Used
Label Encoding:
- Each unique category in a column is assigned a unique integer value.
- Implemented using the `LabelEncoder` class from Scikit-learn.

Example:
Category | Encoded Value  
Red | 0  
Blue | 1  
Green | 2  

## ⚙️ How to Run the Project
1. Clone or download the repository.
2. Open the Jupyter Notebook file.
3. Run the notebook cells sequentially.
4. The notebook will demonstrate Label Encoding on the dataset.

## 🎯 Purpose
Machine learning algorithms require numerical input. Label Encoding is a simple technique to transform categorical variables into numerical values so they can be used in machine learning models.
